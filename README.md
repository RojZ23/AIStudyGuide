# AI Study Guide
This a minimal Spring Boot app with JTE and Spring AI that lets users:
- register / login / logout
- start study chats
- ask questions and get AI answers
- view past conversations
- edit guide
- regenerate guide
- detailed, very detailed, basic generation
- flashcards
- quiz mode
- list page
- analytics page
- forgot password/update password page


## Run
1. Set AI provider and api key
2. `mvn spring-boot:run`
3. Open `http://localhost:8080/`
